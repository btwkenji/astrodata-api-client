# astrodata-api-client

## installation

```sh
git clone https://github.com/kenjitheman/astrodata-api-client
```

- install all dependencies

```sh
yarn install
```

## usage

- run

```sh
yarn start
```

- run in dev mode

```sh
yarn dev
```
